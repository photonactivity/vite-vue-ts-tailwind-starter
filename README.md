#  Vite 2.x + Vue 3.x + Typescript + Tailwind 2.x starter 

> This play project should help get you started developing with Vue 3 Typescript  vuex vue-router Tailwindcss in Vite.

---

## Getting Started

```
git clone https://github.com/photonactivity/vite-vue-ts-tailwind-starter.git
yarn install
yarn dev
```
